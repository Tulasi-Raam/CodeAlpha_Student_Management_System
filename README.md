### Student Management System in C

### Description

This project is a menu-driven Student Management System developed in C as part of the CodeAlpha Internship. It uses structures and file handling to store student records permanently.

### Features

* Add Student

* Display Students

* Search Student

* Update Student

* Delete Student

* File handling for permanent storage

### Technologies Used

* C Programming Language

* Structures

* File Handling

### How to Compile

Compile

Copy

gcc student_management.c -o student_management

### How to Run

Run

Copy

./student_management
